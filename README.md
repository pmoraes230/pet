O **Corações em Patas** é um sistema web desenvolvido para oferecer suporte especializado no cuidado animal. O projeto foca no acompanhamento da saúde de pets, unindo tecnologia e sensibilidade para auxiliar tutores e veterinários na gestão do bem-estar físico e emocional dos animais.

---

## Objetivo do Projeto
Unir tecnologia e empatia para oferecer uma ferramenta que melhore a qualidade de vida dos animais. O sistema promove cuidados físicos e mentais de forma acessível e intuitiva, servindo como um ponto de entrada essencial para o acompanhamento da saúde oncológica.

## Como Rodar o Projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado:
- [Git](https://git-scm.com)
- [Node.js](https://nodejs.org/en/) (ou outra ferramenta necessária)


## Como Executar o Projeto Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

## 1. Clonar o Repositório

Abra o terminal e execute o comando:

Bash

git clone (https://github.com/pmoraes230/pet.git).

## 2. Abrir no VS Code
Acesse a pasta do projeto:

Bash

cd Reposit-rio-cora-o-em-patas-front
code .

## 3. Instalar a Extensão "Live Server"

Para visualizar as alterações em tempo real:

Vá até a aba de extensões no VS Code (Ctrl + Shift + X).

Procure por Live Server.

Clique em Instalar.


## 4. Executar o Projeto

Coloque no terminal o seguinte comando ".\venv\Scripts\python.exe -m daphne -b 0.0.0.0 -p 8000 setup.asgi:application"

O projeto será aberto automaticamente no seu navegador padrão (geralmente em http://127.0.0.1:8000/).
